Install, setup and use OSC Groups to share OSC data in networked performance.

{{< /blocks/cover >}} {{% blocks/section %}} This site shows how to get started with OSC Groups. It explains how to obtain the free binary client applications to use on Windows, Linux or MacOS computers to share data between any kind of OSC enagled application, such as SuperCollider, Max/MSP, Pure Data or other. It also shows how to start the binary application from the terminal in order to connect to a group running a server that will send your data to all other members in this group. Support is provided via email or live online by the authors of this website. Please contact us. {{% /blocks/section %}}
