Learn how to install and share OSC Groups to share data between sound and/or graphics applications in networked performance.

{{< blocks/link-down color="info" >}}
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} OSC Groups can send data from any OSC-capable application to all applications which are connected to a group. In this way, performers can share any type of control data (eg. from MIDI devices ore sensors) With [sc-hacks-redux](https://github.com/iani/sc-hacks-redux), users can send the code which they run locally on their SuperCollider to all other SuperCollider users connected to an OSC Group. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="New chair metrics!" %}} The Sc-Hacks documentation UI now shows chair size metrics by default.
